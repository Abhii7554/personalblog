Understanding JavaScript Closures

By Abhishek Patel | Tags: #JavaScript, #WebDevelopment

Closures are one of the most powerful and often misunderstood features of JavaScript. A closure gives you access to an outer function's scope from an inner function.

In JavaScript, closures are created every time a function is created. Let’s look at a simple example: 


function outer() {
  let counter = 0;
  return function inner() {
    counter++;
    console.log(counter);
  };
}

const increment = outer();
increment(); // 1
increment(); // 2

This works because the inner function keeps a reference to the `counter` variable even after `outer` has finished executing. That’s a closure.

Closures allow for powerful patterns like data privacy, function factories, and more.
